• Tacklr

    "Set your route and Tackl it" goes the motto for Tacklr.
    Tacklr is a native iPhone app built using the Ionic Framework. It's a resource locator and route mapping application with notes, location tagging & photo sharing capabilities. Sort of like a geographic "to-do" list. There are community based file sharing and social networking features planned in future.
    Built with Angular and using the Google Maps API along with Backand remote DB to store profile information. The biggest challenge for this project is the "burn and wipe" functionality that the concept will rely on. I plan to "Tackl" this by allowing users to either call a number and press a certain key, or to have a built-in "panic" button in the app.
    Currently in beta (0.1), Tacklr is a passion project that I continue to build and improve.
