• Spacey: Tonal Gravity

    This is a project where I used Canvas, FabricJS and PhysicsJS to simulate bodies in an outer space environment.
    There is a proximity equation that measures the distance between bodies using Pythagorian priciples, and the force of gravity acting those bodies is represented through sound, via dynamically created oscillators courtesy of Web Audio. Each body creates it's own oscillator, and each oscillator is in turn routed through the custom audio graph. The audio graph contains hand crafted Convolver, Distortion and Delay effect algorithms as well as several preamp gains and Bi-Quad filters to further shape the sound
    I'm also writing a series about the process of constructing these and other Web Audio elements on Medium, including code snippets and walk throughs.
